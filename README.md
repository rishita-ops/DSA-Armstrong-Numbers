# DSA-Armstrong-Numbers
An Armstrong number (also known as a narcissistic number, pluperfect number, or pluperfect digital invariant) is a number that is equal to the sum of its own digits each raised to the power of the number of digits. For example, 153 is an Armstrong number because it has 3 digits, example : 1^3 + 5^3 + 3^3 = 153.
